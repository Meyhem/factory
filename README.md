# Factory CLI

A .NET console application for factory operations.

## Getting Started

### Prerequisites

- .NET SDK 8.0 or later

### Build

```bash
dotnet build
```

### Run

```bash
dotnet run --project Factory.Cli
```

### Clean

```bash
dotnet clean
```

## Solution Structure

- `Factory.sln` - Solution file
- `Factory.Cli/` - Console application project
